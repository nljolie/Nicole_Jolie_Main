Frequently Asked Questions

Is this a retainer model?
Primarily, no. This is a one-time "Build Phase" project to set up the system. After the build, there is an optional light retainer if you want me to review performance, but it is not required.
What are the software costs?
You will pay for the tools directly so you own the accounts. This typically includes n8n, Blotato, Claude, and VidIQ. The combined monthly budget is a few hundred dollars, not thousands.

The build phase is a focused project. Payment is structured as 50% upfront to begin work, and 50% due when the system is complete and in use by your team.
